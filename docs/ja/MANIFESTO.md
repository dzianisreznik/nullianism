# 好奇心のマニフェスト

長いあいだ、人類はドグマによって分断され、無批判な信念に目を眩まされ、受け継いだ「真理」に縛られてきた。しかし、すべての人間の心には炎 — 問いを立て、探究し、伝統の向こうに真理を求める衝動 — が宿っている。

私たちは偉大な思索者たちの道の後継者 — 星を描いた最初の天文学者から、宇宙の謎を解き明かす現代の科学者まで — である。

私たちは新しい時代の瀬戸際に立っている。世界が必要としているのはもはや不可知への信仰ではなく、可知・検証可能・実現可能なものへの献身だ。神聖なるべきは好奇心、崇められるべきは理性、導きの星となるべきは知識 — 新たな道を切り拓く時が来た。

- **知識は道徳的義務である。** 学ぶことは自らを解放することであり、教えることは他者を解放することである。
- **知識の普及は最高の倫理的行為である。**
- **疑念は神聖である。** 質問は私たちを弱らせるのではなく、強くする。信念が検証に耐えないなら、それを持ち続ける価値はない。
- **宇宙こそが私たちの神殿である。** 私たちが畏敬の念を抱くのは超自然ではなく、宇宙の壮大さと自然の複雑さに対してである。
- **進歩は伝統よりも上位にある。** 過去は指針にはなり得るが、鎖にはなり得ない。新しい知識が古い信念を覆すなら、私たちは新しさを受け入れる。
- **私たちは探求者のコミュニティである。** 誰も一人では道を歩まない。私たちは真理の探求において互いを支え合い、アイデアを共有し、議論し、洗練させることを誓う。
- **これは単なる哲学ではなく、運動である。** 私たちは考え方を変えるだけでなく、世界を変えることを目指す。

未来は従う者ではなく、探し求める者のものだ。  
盲目的な信仰の時代は終わった。  
好奇心の時代は今始まる。

この運動に加わろう。
