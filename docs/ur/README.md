# نولیانیت 

> دنیا کا پہلا سائنسی اوپن سورس مذہب۔

نولیانیت کے ریپوزٹری میں خوش آمدید — ایک سائنسی مذہب جو الیگزاندر پانچِن نے تخلیق کیا ہے۔ یہ نہ کوئی مذاق ہے اور نہ ہی کلک-بیٹ، بلکہ ایک سماجی تجربہ اور ایسی قدروں کا نظام تشکیل دینے کی کوشش ہے جو عقائد اور اندھی ایمان پر نہیں بلکہ تجسّس، تنقیدی سوچ اور سائنسی طریقِ کار پر مبنی ہو۔

## بنیادی خیال

نولیانیت ایسا مذہب ہے جس کا مقصد لوگوں کو کسی معبود یا متعدد معبودوں کے گرد نہیں بلکہ انسانی تجسّس کے گرد متحد کرنا ہے۔ ہم مذہب کے “خول” — خیالات کو پھیلانے اور برادریاں بنانے کے اُس طاقتور نظام — کو استعمال کرتے ہیں، مگر اُسے عقلی اور انسان دوست مواد سے پُر کرتے ہیں۔

**ہمارا نشان — صفر (0)**۔ یہ علامت ہے:

- عقائد اور تعصبات کی عدم موجودگی کی۔
- علم کی نقطۂ آغاز کی۔
- تمام شرکائے کار کا مرکز سے یکساں فاصلے پر ہونا، یعنی کوئی درجہ بندی نہیں۔
- عقل کی وہ انگوٹھی جو لوگوں کو جوڑتی ہے۔

## نولیانی کیسے بنا جائے؟

اگر آپ یہاں بیان کردہ خیالات سے اتفاق رکھتے ہیں تو سمجھ لیجیے کہ آپ پہلے ہی نولیانی ہیں۔ کسی رسمی کارروائی کی ضرورت نہیں۔ ہمارا مذہب غیر مرکزی (decentralized) ہے۔
